# 🏀 SwishOps

An AI-powered Basketball Insights Platform built to demonstrate DevOps mastery.

## Tech Stack
- **Cloud:** AWS (EKS, RDS, Lambda, ECR, Route53)
- **IaC:** Terraform + Ansible
- **CI/CD:** Jenkins
- **Containers:** Docker + Kubernetes + Helm
- **Monitoring:** Prometheus + Grafana
- **Backend:** FastAPI (Python)
- **Frontend:** React
- **AI:** OpenAI + fallback providers
